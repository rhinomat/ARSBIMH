# ARSBIMH
## Description
The ARSBIMH is the Automated Retrieval System for Business Information Meant for Hiring\
It retrieves the list of Oregon Businesses from the Oregon Data Repository and converts it into an Open Document Spreadsheet for use with Libreoffice Calc.\
This project will help with filtering out companies in Oregon that would be geographically desirable to work for.
## Tools Used for Building
* Python
* UV (For constructing the project)
* Pywright (For retrieving the information needed by maneuvering the desired webpages)
* Pandas (For data manipulation)
* odfpy (For interfacing with the Spreadsheet (Libreoffice Calc uses Open Document Spreadsheets))
